## Setup

- Required ENV Variables:

  ```sh
  API_URL='http://localhost:5001/graphql'
  ```

- Running the project:

  ```sh
  cd coolmovies-backend
  docker-compose up -d
  cd ../coolmovies-frontend
  npm run dev
  ```
